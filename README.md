# Disease-Prediction-Django-
# Disease Prediction System 🏥🤖

An AI-powered healthcare web application built using **Django** and **Machine Learning** that predicts diseases based on user symptoms and medical report files.  
The system provides an interactive modern UI, prediction history tracking, and intelligent disease analysis.

---

# 🚀 Features

## ✅ AI Disease Prediction
Predict diseases based on symptoms selected by the user using Machine Learning models.

## ✅ File-Based Prediction
Upload medical CSV reports for automated disease prediction.

## ✅ Prediction History
Stores and displays previous prediction records with an interactive dashboard.

## ✅ Modern UI/UX
- Responsive Design
- Glassmorphism Effects
- Animated Components
- Interactive Dashboard
- Modern Healthcare Theme

## ✅ Smart Analytics Dashboard
Displays:
- Total Predictions
- AI Monitoring
- Healthcare Statistics

## ✅ Fully Responsive
Works on:
- Desktop
- Tablet
- Mobile Devices

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/Disease-Prediction-System.git
```

---

## 2️⃣ Open Project Folder

```bash
cd Disease-Prediction-System
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate Virtual Environment

### Windows
```bash
venv\Scripts\activate
```

### Linux/Mac
```bash
source venv/bin/activate
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 6️⃣ Run Migrations

```bash
python manage.py migrate
```

---

## 7️⃣ Start Server

```bash
python manage.py runserver
```
